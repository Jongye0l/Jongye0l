# 컴퓨터의 아버지 찰스 배비지
### 개요
우리 생활속에서 매일 사용하고 있고 이제는 없어서 안되는 내가 제일 관심있어하는 컴퓨터를 누가 처음 개발하게 되었는지 궁금하고
제가 원하는 직업인 프로그래머는 컴퓨터에 대해 더 자세히 알 필요가 있다는 생각이 들어 '컴퓨터의 아버지'라고 불리는 찰스 배비지를 선정하게 되었습니다.

### 본론
찰스 배비지는 1791년 12월 26일에 태어나게 되었고 부유한 아버지 덕에 여러 학교와 여러 가정교사 밑에서 초등교육을 받을 수 있었다. 찰스 배비지는 1810년 케임브리지 대학교 트리니티 칼리지에 입학했다.
